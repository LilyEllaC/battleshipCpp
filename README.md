# battleshipCpp

## Includes

*   **ASCII Images** the positions of your boats and all shots are shown on the grid
*   **1 player** It is a game against a computer that is reasonably smart, but not much better then the average person


## Installation

To play the game of battleship, you need to download and run the executable. 
*Important* the executable is made for Linux, and you need a terminal or command prompt application to run it in

### Running the game

1. *Downloading* Get the executable from the directly from the itch.io page (https://lilyec.itch.io/battleship)

2. *Navigating* Open your terminal and go to the directory where you want to download it. 

3. *Executing* Run the game using the correct command for the system
*   **Linux** 
    ```bash
    .\battleship
    ```

## How to play
*   Follow the instructions
*   Choose which boat to place, then choose the starting and ending coordinates
*   Repeat previous step until all boats are placed. 
**Important** boats cannot go diagonally or on top of each other
*   Start shooting at the computers boat by entering te coordinates
*   If you hit a boat it will say 'HIT!' and will show the hit in a different colour
*   When a boat is sunk it will both say so, and have it shown beside the board
*   The game ends when you have sunk all of the computer's boats, or they sunk all of yours
